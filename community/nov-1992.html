As usual, this is distributed in plain text form, but the original hypertext contains lots of links and may be read as original news page. If you don't have World-Wide Web software yet, you can visit CERN’s information system at info.cern.ch.

Client software

Three developments on the client side. Tony Johnson of Boston University, developer of the MidasWWW browser for Motif, has ported it now to four X11 platforms and has a tar file of sources available on: CERN information system or FreeHEP archive.

Here at CERN, Nicola Pellow is back until the end of the year, and has picked up the Mac Browser. She has a pre-alpha with basic functionality up.

The full-screen client (using curses) has been released by Jim Whitescarver of NJIT. See release note for details.

The NeXTStep client has been revised. The 0.14 version is recommended for users. Binary available via CERN FTP archive.

More and more hypertext online

New W3 servers have appeared at KVI, CWI, IN2P3, and NCSA.

CWI has a hypertext version of GNU documentation and a guide to Audio formats. NCSA has hypertext documentation for the X-Collage system. Two CERN projects include ADAMO and RD13.

Meanwhile, Cornell Law School has a server with hypertext of US Copyright Law.

Browse the WAIS servers

It has been difficult browsing WAIS sources. Now, looking under "types of server" on the web leads to hypertext lists of WAIS servers organized by name and domain. Currently there are many databases available across multiple hosts.
